## 使用
安装weget curl依赖包<br>
`yum update -y && yum install curl -y`   CentOS/Fedora<br>
`apt-get update -y && apt-get install curl` -y   Debian/Ubuntu<br>
远程下载代码<br>
`curl -sS -O https://raw.githubusercontent.com/ecouuc/shell/main/eco.sh && sudo chmod +x eco.sh && ./eco.sh`
